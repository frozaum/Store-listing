Essa extensão dá a possibilidade para usar seus controles de mídia para controlar a reprodução desses serviços.
Quando você começa a assistir/ouvir algum serviço que é suportado pela extensão, a extensão envia a metadata do(a) vídeo/música para o nosso cliente de desktop e o PreMiD o processa. O PreMiD envia a informação com a forma correta para o Discord e automaticamente define seu status depois.
